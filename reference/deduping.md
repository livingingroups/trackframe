# Deduping of trackframe

This function removes duplicated entries from objects of class
trackframe.

## Usage

``` r
deduping(tf, cols = tf_colnames(tf))
```

## Arguments

- tf:

  an object of class trackframe

- cols:

  determines the unique identifier. Default are the key columns provided
  by tf_colnames(tf).

## Value

an object of class trackframe
