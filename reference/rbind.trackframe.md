# Combine trackframes by rows

Take a sequence of trackframes to combine rows.

## Usage

``` r
# S3 method for class 'trackframe'
rbind(...)
```

## Arguments

- ...:

  objects of class trackframe.

## Value

an object of class trackframe

## Examples

``` r
tf1 <- trackframe::tf_mini
tf2 <- tf1
tf2$id <- c(rep("A",5), rep("B", 4), rep("C",2))
tf2$northing <- 20
tf2$easting <- 10
tf1_tf2 <- rbind(tf1, tf2)
```
